My CSS Box Model Demo

About This Page

This simple demo shows how the CSS box model works with margins, borders, padding, and content.

Every element on a webpage can be thought of as a rectangular box. The spacing, sizing, and alignment you see are controlled through the box model.

Another Example

Notice how spacing and layout are controlled with the margin, padding, and border properties.

Margins add space outside an element, while padding adds space inside between the content and the border. This distinction is crucial for precise layout design.
